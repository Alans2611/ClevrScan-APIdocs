---
title: "Delete Template"
api: "DELETE https://api.clevrscan.com/api/templates/{templateId}"
description: "Deletes an existing template by its ID. Once deleted, the template cannot be recovered."
authMethod: key
---

### Request Format

**HTTP Request**

```http
DELETE /api/templates/{templateId}
x-api-key: <api_key>
```

### Parameters

| Parameter  | Type   | Required | Description                         |
| ---------- | ------ | -------- | ----------------------------------- |
| templateId | String | Yes      | Unique ID of the template to delete |
