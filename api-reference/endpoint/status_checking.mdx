### Request Format

```http
GET /api/status/celery-task-uuid
x-api-key: <api_key>

```

<RequestExample>
```bash cURL
curl -X GET "https://api.clevrscan.com/api/status/<task_id>" \
  -H "x-api-key: <api_key>"
```
