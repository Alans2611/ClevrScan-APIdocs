---
title: "Get Document History"
api: "GET https://api.clevrscan.com/api/document_history"
description: "Fetches the history of uploaded and processed documents with pagination support."
authMethod: key
---

### Request Format

**HTTP Request**

```http
GET /api/document_history?page={page}&limit={limit}
x-api-key: <api_key>
```

### Parameters

| Parameter | Type   | Required | Default | Description                               |
| --------- | ------ | -------- | ------- | ----------------------------------------- |
| page      | Number | No       | 1       | Page number of results to retrieve        |
| limit     | Number | No       | 15      | Number of results per page                |

<RequestExample>
```bash cURL
curl 'https://api.clevrscan.com/api/document_history?page=1&limit=15' \
  -H 'Accept: application/json, text/plain, */*' \

  -H 'x-api-key: <api_key>' \
  -H 'Connection: keep-alive' \
```
