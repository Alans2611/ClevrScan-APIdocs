### Path Parameters

| Parameter  | Type   | Required | Description                         |
| ---------- | ------ | -------- | ----------------------------------- |
| documentId | Number | Yes      | Unique ID of the document to update |

| gpt\_result | Object | Yes      | Object containing updated AI results or metadata  |
